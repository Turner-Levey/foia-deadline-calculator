# FOIA 20 Working Day Deadline Calculator

Free no-tracking FOIA due-date calculator for federal public-records request tracking:

- Estimate the 20-working-day federal FOIA determination target from a clock-start date.
- Exclude Saturdays, Sundays, and standard U.S. federal holidays.
- Add optional tolling / pause days and an optional 10-working-day unusual-circumstances extension target.
- Copy a source-linked memo, copy or download a CSV row, and download an `.ics` calendar hold.
- Use custom closure dates for emergency federal closures or agency-specific date handling.

The tool runs entirely in the browser. It has no signup, cookies, analytics, beacons, external scripts, uploads, or browser storage. Calendar export creates a local `.ics` planning hold from the calculated target date.

Live site: https://foia-deadline-calculator.vercel.app/
